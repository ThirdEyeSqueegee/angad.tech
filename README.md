## A personal website built with [Vite](https://vitejs.dev), [Joy UI](https://mui.com/joy-ui/getting-started), and [Motion](https://motion.dev)

### Prerequisites

- [pnpm](https://pnpm.io/)
- [Node.js](https://nodejs.org)

### Setup

1. Clone the repository
2. Run `pnpm i`

### Development

- Run `pnpm dev`

### Build

- Run `pnpm build`
